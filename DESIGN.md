# DESIGN.md

## Design Philosophy

The website should communicate:

* Trust
* Professionalism
* Adventure
* Luxury
* Safety

Visitors must immediately feel:

"This company looks reliable enough to trust with my vacation."

The design should resemble premium international travel brands rather than generic local tour websites.

Examples of inspiration:

* Airbnb Experiences
* GetYourGuide
* Klook
* Booking.com Activities
* Intrepid Travel

Avoid:

* Cheap template appearance
* Excessive animations
* Cluttered layouts
* Too many colors
* Stock-template aesthetics

# Animations

Use subtle animations only.

Allowed:

Fade In

Slide Up

Scale In

Image Hover Zoom

Do NOT use:

Spinning

Bounce

Flash

Overly dramatic effects

---

# Mobile Design

Must feel native.

Requirements:

* Hamburger menu
* Large tap targets
* Minimum 16px text
* No horizontal scroll
* Optimized image loading

---

# Trust Elements

Display throughout website:

✓ Google Reviews

✓ Licensed Tour Operator

✓ Secure Payment

✓ Local Expert Guide

✓ 24/7 Support

✓ Best Price Guarantee

---

# Photography Style

Use images that show:

* Beaches
* Mountains
* Adventure activities
* Happy travelers
* Luxury accommodations
* Local culture

Avoid:

* Low quality images
* Generic stock photos
* Over-edited imagery

Images should feel authentic.

---

# Conversion Optimization

Every section should encourage booking.

Always include:

* CTA buttons
* WhatsApp shortcuts
* Inquiry forms
* Social proof

The visitor should never need to search for how to contact the company.

---

# Final Goal

Create a travel website that feels like a premium international tour operator.

Users should trust the company within the first 5 seconds of visiting the website.

The website should prioritize conversion, credibility, and mobile usability above all else.
