# AGENT.md

## Project

Build a premium Travel & Tour Agency website designed to generate bookings and inquiries from both domestic and international travelers.

This project must strictly follow:

* DESIGN.md
* SKILL.md

If any conflict occurs:

SKILL.md > DESIGN.md > AGENT.md

---

# Business Objective

Primary Goal:

Convert website visitors into customers.

Conversion methods:

* WhatsApp Booking
* Contact Form
* Quote Request

Success metrics:

* More inquiries
* More bookings
* Better credibility
* Better Google visibility

---

# Target Audience

## Primary

International Tourists

Characteristics:

* Looking for local experiences
* Mobile-first users
* Trust and safety focused
* Compare multiple operators before booking

---

## Secondary

Domestic Travelers

Characteristics:

* Looking for affordable packages
* Family vacations
* Group tours
* Honeymoon packages

---

# Website Type

Marketing Website

Travel Agency Landing Page

Not an e-commerce website.

Not a marketplace.

Not a travel OTA.

Focus on lead generation.
